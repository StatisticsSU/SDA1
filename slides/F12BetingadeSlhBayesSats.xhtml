• Bayes sats P( B|A ) = P( A | B ) P( B ) P( A | B ) P( B ) +P( A | B c ) P( B c )
• Covid: A = {pos}. B = {covid} P( covid | pos ) = P( pos | covid ) P( covid ) P( pos | covid ) P( covid ) +P( pos | inte covid ) P( inte covid )
• Notera: P( neg | inte covid ) =1-P( pos | inte covid ) .
• Prevalens: P( covid ) - andel med covid i populationen.
• Sensitivitet: P( pos | covid ) - hur känsligt är testet för att upptäcka covid?
• Specificitet: P( neg | inte covid ) - är testet specifikt för covid, eller reagerar det även på annat?
