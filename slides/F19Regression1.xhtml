• Residualvariansen - hur bra regressionslinjen passar data: s e 2 = ∑ i=1 n ( y i - y ˆ i ) 2 n-2
• Kom ihåg: stickprovsvariansen delar med n-1 eftersom vi måste beräkna y ‾ först: s y 2 = ∑ i=1 n ( y i - y ‾ ) 2 n-1
• Residualvariansen delar med n-2 eftersom vi måste beräkna både b 0 och b 1 först. Väntevärdesriktig.

• Hälsobudgetdata s e 2 = 76.056 29-2 ≈ 2.817 s e = 2.817 ≈ 1.678  år

Frame
• Populationsmodell för enkel regression: Y= β 0 + β 1 x+ ϵ , ϵ ∼ N( 0, σ ϵ 2 )
• β 0 är interceptet i populationen/modellen.
• β 1 är lutningen på regressionslinjen i populationen.
• Regressionlinjen i populationen är ett betingat väntevärde: E( Y | x ) = β 0 + β 1 x
• β 1 : hur Y förändras i genomsnitt när x ökar med en enhet.
• “i genomsnitt” = (betingat) väntevärde.
• Responsvariabeln y kommer avvika från populationens regressionslinje med en slumpmässig “felterm” ϵ .

• Populationsmodell för hela stickprovet: Y i = β 0 + β 1 x i + ϵ i , ϵ i ∼ N( 0, σ ϵ )
• Stickprov/datamaterial med n observationspar ( y 1 , x 1 ) , … ,( y n , x n )
• I regression antar vi att x -variabeln inte är slumpmässig.
